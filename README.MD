## **Análise das Avaliações dos Clientes**

Este repositório concentra um estudo sobre análise de avaliações de clientes que foi desenvolvido com o objetivo de aplicar conceitos microeconômicos, técnicas de análise e ciência de dados utilizando Python e SQL. O projeto foi elaborado a partir do tema do meu Trabalho de Conclusão de Curso (TCC) que emprega o modelo Logit a fim de explorar fatores que influenciam as avaliações dos clientes e apoiar a tomada de decisão.
