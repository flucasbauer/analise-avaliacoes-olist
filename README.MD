# Análise das Avaliações dos Clientes

Este repositório reúne um estudo sobre análise de avaliações de clientes, desenvolvido com o objetivo de aplicar conceitos microeconômicoss e técnicas de análise e ciência de dados utilizando Python e SQL. O projeto foi elaborado a partir do tema do meu Trabalho de Conclusão de Curso (TCC) que empregha o modelo Logit a fim de explorar fatores que influenciam as avaliações dos clientes e transformar dados em informações para apoio à tomada de decisão.
